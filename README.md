## 🚀 Déploiement sur GitHub Pages

1. Créer un repo nommé `oralE5` sur GitHub
2. Pousser le contenu de ce dossier
3. Aller dans `Settings > Pages`
4. Source : `Deploy from a branch` → branche `main` → dossier `/ (root)`
5. Le site sera accessible à : `https://vacbard91.github.io/oralE5/`

## 📂 Ajout des documents

Placer les fichiers de preuve (PDF, PNG, TXT, ZIP) dans les dossiers correspondants :
- `docs/formation/` pour les réalisations en formation
- `docs/pro-annee1/` pour la 1ère année pro
- `docs/pro-annee2/` pour la 2ème année pro

Les noms de fichiers référencés dans le code HTML doivent correspondre exactement à ceux placés dans ces dossiers.
